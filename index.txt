===================
Pyswisseph Homepage
===================

-----------------------------------------------
Python extension to AstroDienst Swiss Ephemeris
-----------------------------------------------

A very useful module for researchers in astrology!

A number of applications have been written with it, including:

  - `OpenAstro.org`_, a native Ubuntu software by Pelle van der Scheer
  - `AstroWS`_, a REST web service by Jean-AndrÃ© Santoni
  - `Oroboros`_, by myself... actually unmaintained, but had its hour of
    glory in `Linux Format`_ magazine of October 2008, yay!

License:

  - Either `GNU Public License`_ v2 or later
  - or `Swiss Ephemeris Professional License`_

Usage example:

>>> import swisseph as swe
>>> help(swe)

Installations with distutils:

``python setup.py install``

Download
--------

  - Sources are at the `Python Package Index`_
  - Precompiled binaries for Windows are available for a small fee. `Contact me`_.
